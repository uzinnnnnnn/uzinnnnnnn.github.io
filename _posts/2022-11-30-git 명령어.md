#### 현재 작업중인 디렉토리를 git 저장소로 지정
`git init`  

#### Git에 변경사항 반영하기
`git add example.py`
example.py를 생성(혹은 수정)하고, 이를 Commit에 반영하고 싶은 경우

#### commit 기록 확인하기
`git log`

#### Branch 생성
`git branch <branch_name>`

#### Branch 병합
`git merge <branch_name>`
